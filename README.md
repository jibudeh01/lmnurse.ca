LMNurse Platform

A full-stack web platform for LMNurse, designed to support healthcare staffing and professional registration workflows.

Live site: https://lmnurse.ca

Overview

This project goes beyond a static website and implements a dynamic system for collecting, processing, and managing user registration data. It reflects real-world application development with both frontend and backend components.

Tech Stack

HTML5

CSS3

JavaScript

Node.js

MongoDB

Amazon EC2 (Ubuntu)

Plesk (frontend hosting and management)

Features

Dynamic registration and form handling

Backend API for data processing and storage

Database integration with MongoDB

Multi-component architecture (frontend + backend)

Production-ready deployment

Deployment

The frontend is hosted via Plesk, while the backend services and database run on an Ubuntu-based EC2 instance.

Notes

This project represents a full, production-grade system, demonstrating backend development, database design, and multi-service deployment rather than simple static content delivery.
